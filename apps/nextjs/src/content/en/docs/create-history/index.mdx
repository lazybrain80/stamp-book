---
title: Watermark Creation History
description: This screen shows the history of created watermarks.
---

## Initial Screen 🖥️
![init](/images/docs/en/c-history/init.png)

You can check the history of recently created watermarks. Each item displays the creation date and time, and the content of the created watermark.

## Load More History Data 🔄
![loadmore](/images/docs/en/c-history/tail.png)

If there is a lot of history data, you can click the **"Load More"** button to load additional data. Clicking this button will allow you to view more creation history.

<Callout>
📢 **Note** The watermark creation history feature is currently under development, so sometimes the data may not be displayed correctly. If you encounter any issues, please contact the administrator.
</Callout>

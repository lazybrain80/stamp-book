---
title: Watermark Validation History Screen
description: This screen shows the history of validated watermarks.
---

## Initial Screen 🖥️
![init](/images/docs/en/v-history/init.png)

You can check the history of recently validated watermarks. Each item displays the validation date and time, and the validation result.

## Load More History Data 🔄
![loadmore](/images/docs/en/v-history/tail.png)

If there is a lot of history data, you can click the **"Load More"** button to load additional data. Clicking this button will allow you to view more validation history.

<Callout>
📢 **Note** The watermark validation history feature is currently under development, so sometimes the data may not be displayed correctly. If you encounter any issues, please contact the administrator.
</Callout>

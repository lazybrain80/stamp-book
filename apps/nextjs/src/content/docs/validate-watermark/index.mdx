## 이미지 선택 🖼️
![selection](/images/docs/validation/image_selection.png)

여기서는 ** 워터마크가 적용된 이미지 **를 선택합니다. 이미지를 선택하면 미리보기 화면에 선택한 이미지가 표시됩니다.
추가로 아래에 입력창에 확인하고자 하는 ** 워터마크 **를 입력합니다.
입력 후 ** "검증하기" ** 버튼을 클릭하여 워터마크를 검증합니다.
